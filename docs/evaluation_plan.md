# pWPS Agent 评估计划 evaluation_plan.md

## 1. 文档定位

本文档定义系统质量评估指标、评估集、人工评审流程和持续优化方式。

---

## 2. 评估目标

评估系统是否能稳定生成可审查的 pWPS 草案，而不是评估模型是否能写出漂亮文本。

---

## 3. 核心指标

### 字段抽取

```text
基础字段抽取准确率
字段单位解析准确率
字段缺失识别率
禁止推断字段违规率
```

### 候选生成

```text
候选合理率
证据覆盖率
高风险字段风险提示率
候选与上下文一致性
```

### 审计

```text
焊法-焊材冲突检出率
元信息伪造检出率
参数缺失检出率
高风险字段低证据检出率
```

### 用户体验

```text
Guided 平均确认轮次
用户接受推荐比例
用户查看证据比例
移动端核心流程完成率
```

### 系统稳定性

```text
Workflow 成功率
中断恢复成功率
Schema validation 失败率
Skill retry 次数
```

---

## 4. 评估集设计

至少包含：

```text
正常样例
缺失字段样例
焊法焊材冲突样例
高风险热处理样例
元信息伪造风险样例
用户覆盖建议样例
低证据样例
```

---

## 5. 人工评审表

人工评审关注：

```text
字段是否合理
证据是否对应字段
风险是否充分
是否存在编造
审计是否发现明显问题
草案是否可作为讨论材料
```

---

## 6. 版本对比

每次评估记录：

```text
code version
schema_version
field_registry_version
workflow_version
skill_versions
prompt_versions
model versions
```

这样才能比较不同版本系统质量。
